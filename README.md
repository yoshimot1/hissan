# hissan (ひっさん)

子供向けの2桁-1桁の引き算計算ゲームアプリです。

## 機能

- 🎲 2桁(10-99) - 1桁(1-9) のランダムな引き算問題
- 🎨 子供向けのカラフルで分かりやすいUI
- 🎉 正解時の華やかな演出（紙吹雪アニメーション）
- 📊 10問1セット
- ⏱️ タイマー機能で所要時間を計測
- 📈 最終結果表示（正解数と所要時間）

## 使い方

1. `index.html` をブラウザで開く
2. 「スタート」ボタンをクリック
3. 表示された問題に答えを入力
4. 「こたえる」ボタンをクリック（またはEnterキーを押す）
5. 10問終了後、結果が表示されます

## 必要環境

- モダンなウェブブラウザ（Chrome, Firefox, Safari, Edgeなど）
- インターネット接続は不要（完全にオフラインで動作）
