# はなちゃんのけいさんアプリ

子供向けの引き算計算ゲームアプリです。

## 機能

- 🎮 3つの難易度モード
  - **Easy**: 20より小さい数字の2桁(10-19) - 1桁(1-9)
  - **Normal**: 2桁(10-99) - 1桁(1-9)
  - **Hard**: 2桁(10-99) - 2桁(10-99)
- 🎨 子供向けのカラフルで分かりやすいUI
- 🎉 正解時の華やかな演出（紙吹雪アニメーション）
- 📊 10問1セット
- ⏱️ タイマー機能で所要時間を計測
- 📈 最終結果表示（正解数と所要時間）

## 使い方

1. `index.html` をブラウザで開く
2. 「スタート」ボタンをクリック
3. 難易度（Easy、Normal、Hard）を選択
4. 表示された問題に答えを入力
5. 「こたえる」ボタンをクリック（またはEnterキーを押す）
6. 10問終了後、結果が表示されます

## 必要環境

- モダンなウェブブラウザ（Chrome, Firefox, Safari, Edgeなど）
- インターネット接続は不要（完全にオフラインで動作）
